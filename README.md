### uni-shop 微信小程序

#### 一、目录结构

- uni-shop								项目主目录
  - components 				 组件文件夹
    - goods-item			 商品组件
    - search                     搜索功能组件
  - pages                              pages页面文件夹
    - cart                          购物车页面
    - cate                         分类页面
    - home                      首页页面
    - profile                     我的页面
  - static                               静态文件文件夹
  - subpkg                            subpkg分包文件夹
    - goods_detail           商品详情页
    - goods_list               商品列表页
    - search                     搜索页面
  - App.vue                          APP入口文件
  - index.html                      页面入口文件
  - main.js                            Vue入口js文件
  - manifest.json
  - package.json
  - package-lock.json
  - pages.json
  - README.md
  - uni.scss

#### 二、项目简介

通过uni-app开发的微信小程序，前端使用Vue框架，按黑马程序员微信小程序教程编写，教程地址：https://www.bilibili.com/video/BV1834y1676P?p=1&vd_source=746b25cf30de2632a3f6d15e93c6c910
